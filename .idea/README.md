# [Gestion de torneos de Fútbol] API

> API REST robusta para la administración y gestión de ligas y torneos de fútbol.

Este backend permite la gestión integral de entidades deportivas (Jugadores, Entrenadores, Estadios), programación de partidos y seguimiento de estadísticas y resultados en tiempo real. Diseñado con arquitectura escalable utilizando Java y Spring Boot.
